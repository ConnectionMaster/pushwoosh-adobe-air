Schedules local notification

onDeviceReady

public function onDeviceReady():void

Call this after all the callbacks have been set

Example

<code> var pushwoosh:PushNotification = PushNotification.getInstance(); pushwoosh.addEventListener(PushNotificationEvent.PERMISSION_GIVEN_WITH_TOKEN_EVENT, onToken); pushwoosh.addEventListener(PushNotificationEvent.PERMISSION_REFUSED_EVENT, onError); pushwoosh.addEventListener(PushNotificationEvent.PUSH_NOTIFICATION_RECEIVED_EVENT, onPushReceived); pushwoosh.addEventListener(PushNotificationEvent.PUSH_NOTIFICATION_OPENED_EVENT, onPushOpened);

pushwoosh.onDeviceReady();

pushwoosh.registerForPushNotification(); </code>

setUserId

public function setUserId(userId: String):void

Set User indentifier.  This could be Facebook ID, username or email, or any other user ID.  This allows data and events to be matched across multiple user devices.

postEvent

public function postEvent(eventId: String,
attributes: Object):void

Post events for In-App Messages.  This can trigger In-App message display as specified in Pushwoosh Control Panel.

Example
